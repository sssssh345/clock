# 📝 To-Do List

A simple and clean to-do list web app to help you organize your tasks and stay productive throughout the day.

## ✨ Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive design
- Saves tasks using browser's local storage (no backend required)

## 🛠️ Tech Stack

- **HTML**
- **CSS**
- **JavaScript**

## 🚀 How to Run the Project

1. Clone or download the repository.
2. Open the `index.html` file in your browser.
3. Alternatively, if you're using VS Code:
   - Right-click on `index.html` and select **"Open with Live Server"**.
