# 🌍 World Clock

A sleek and functional web-based world clock that displays the current time in multiple global cities. Perfect for remote teams, international collaboration, or just checking what time it is around the world.

## ✨ Features

- Displays current time in various major cities (e.g., New York, London, Tokyo, Delhi)
- Real-time clock updates
- Clean, responsive UI
- Lightweight and fast – no backend required

## 🛠️ Tech Stack

- **HTML**
- **CSS**
- **JavaScript**

## 🚀 How to Run the Project

1. Clone or download the repository.
2. Open the `index.html` file in your browser.
3. Or, in VS Code:
   - Right-click on `index.html` and select **"Open with Live Server"**.
